# OMR 카드 자동 채점 + 엑셀 점수 반영

PPT로 만든 OMR 카드를 **볼펜으로 색칠**한 뒤 스캔/촬영한 이미지에서 답을 읽고, 정답과 비교해 채점한 다음, 엑셀의 **해당 사번** 행에 점수를 넣는 도구입니다.

## 가능 여부

- **OMR 카드 읽기**: 가능합니다. 스캔/사진 이미지에서 버블(원) 영역을 찾고, 각 영역이 **볼펜으로 색칠되었는지** 픽셀 밝기로 판단합니다.
- **볼펜**: 검정/파랑 볼펜은 충분한 대비로 인식됩니다. 가능하면 **검정 볼펜** 사용을 권장합니다.
- **사번 매칭**:  
  - 카드에 **QR코드**가 있으면 `--use-qr` 옵션으로 이미지에서 사번을 읽을 수 있습니다.  
  - 없으면 **파일명**에 사번을 넣어 두세요. (예: `12345.png`, `OMR_12345.jpg`)

## 설치

```bash
cd omr-grader
pip install -r requirements.txt
```

## 설정

1. `config_example.yaml`을 복사해 `config.yaml`로 저장합니다.
2. **정답 키**를 실제 시험 정답으로 수정합니다. (40문항, 각 1~5번 중 하나)
3. 엑셀 시트의 **사번 컬럼 이름**이 다르면 `excel.id_column`을 수정합니다.
4. OMR 카드 레이아웃이 40문항 5지선다가 아니면 `layout`을 수정합니다.

## 사용 방법

### 1) OMR 이미지에서 답만 읽기 (테스트용)

```bash
python read_omr.py "경로/OMR카드이미지.jpg"
# 문항 수/선택지 수가 다르면
python read_omr.py "경로/OMR카드이미지.jpg" 40 5
```

### 2) 여러 장 채점 후 엑셀에 점수 반영

```bash
# 파일명에 사번이 있을 때 (예: 12345.png, 67890.png)
python grade_and_excel.py --excel "QR코드양식_학생명단올리기(1급 승진예정자).xlsx" image1.png image2.png ...

# 카드에 QR코드가 있어서 사번을 QR로 읽을 때
python grade_and_excel.py --excel "응시생명단.xlsx" --use-qr scan1.jpg scan2.jpg ...

# 먼저 결과만 보고 싶을 때 (엑셀 저장 안 함)
python grade_and_excel.py --excel "명단.xlsx" --dry-run *.jpg
```

- `--config`: 설정 파일 경로 (기본 `config.yaml`)
- `--dry-run`: 채점만 하고 엑셀에 저장하지 않음
- `--use-qr`: 각 이미지에서 QR코드를 읽어 사번으로 사용

### 3) (추천) 시트1에 시험번호/시험정답 관리, 시트2에 응시생 명단

엑셀 하나로 여러 시험을 관리하고 싶다면, 다음처럼 구성할 수 있습니다.

- **시트1**  
  - A열: `시험번호`  (예: `2024-01`, `2024-02` …)  
  - B열: `시험정답`  (예: `12341234...` 또는 `1 2 3 4 ...` 처럼 40개의 1~4 숫자)
- **시트2**  
  - `이름 / 소속(학교) / 학번(오직 숫자만 가능) / 수강신청 과정 / 점수` (지금 PDF 양식처럼)

이렇게 만든 뒤, 시험마다 **시험번호만 바꿔서** 다음처럼 실행하면 됩니다.

```bash
# 예: 시험번호가 2024-01 인 시험 채점
python grade_and_excel.py --excel "응시생명단.xlsx" --exam-code "2024-01" scan1.jpg scan2.jpg ...
```

- 스크립트가 **시트1**에서 `시험번호=2024-01` 인 행을 찾아 `시험정답`을 읽어 정답 키로 사용합니다.
- `--exam-code`를 사용하면, 기본적으로 **시트2**를 응시생 명단 시트로 보고 그곳에 `점수`를 기록합니다.

## 동작 요약

1. **read_omr.py**: 이미지에서 버블 위치를 찾고, 각 버블이 “색칠됐는지” 판단해 문항별 선택 번호(1~5) 리스트 반환.
2. **grade_and_excel.py**:  
   - 위 결과와 `config.yaml`의 정답 키로 채점.  
   - 사번은 QR 또는 파일명에서 추출.  
   - 엑셀에서 해당 사번 행을 찾아 `점수` 컬럼에 기록.

## 주의사항

- **촬영/스캔 품질**: 카드가 너무 기울어지거나 흐리면 인식률이 떨어질 수 있습니다. 가능하면 스캐너 사용을 권장합니다.
- **레이아웃**: 현재는 “버블을 찾아 위치로 정렬한 뒤, 순서대로 1번 문항 1~5, 2번 문항 1~5, …” 로 가정합니다. PPT OMR 레이아웃이 다르면 `read_omr.py`의 정렬/그룹 로직을 조정해야 할 수 있습니다.
- **정답 키**: 반드시 `config.yaml`의 `answer_key`를 실제 시험 정답으로 수정한 뒤 사용하세요.

이제 OMR 카드를 제출받은 뒤 이미지로 저장하고, 위 순서대로 실행하면 해당 사번의 응시생에게 점수가 자동으로 매겨져 엑셀에 반영됩니다.
